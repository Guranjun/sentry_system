# 基于IMX6ULL的智能哨兵系统(v2.0)
## 核心库说明
### 1.ffmpeg
    实现ffmpeg的视频封装功能，将待存储的图像数据封装成.avi文件或.mp4文件，是基于时间戳的自适应帧率的，方便日后如果接入音频的话进行同步
    说明一下，并没有进行h.264编码，原因是开发板算力应该不够，不过可以期待一下实现h.264软编测试一下效果
### 2.sqlite3
    利用sqlite3实现数据库的操作，利用数据库存储日志，日志操作还是非常重要的，便于调试。
    不过整个数据库上传至上位机的这个操作还没有实现，这是下一个版本的核心目标之一
### 3.opencv
    通过opencv库实现了简单的帧差法，利用帧差法实现环境的监测。具体实现在image_process.cpp文件里，我只写了简单的move_detect的功能，如果想实现更复杂的功能可以在这个文件里写函数，在alarm.cpp里的move_detect位置调用就好。
## 系统整体功能说明
    利用v4l2进行摄像头数据采集，利用自研消息队列发送数据，采用生产者-消费者架构。图像数据的相关操作有udp上传至上位机显示，告警模块的图像处理相关，存储模块的视频存储。
